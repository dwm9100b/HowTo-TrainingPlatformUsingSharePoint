Now that our SharePoint site has been created, its time to create our lists.
Our first list will hold the listing of Courses.

  1. If not already there, navigate to the Training site
  2. From the upper toolbar, click on +New, select List
  3. Create a new Blank list
  4. Name: Courses
  5. Click on Create
  6. Once created, go into Settings (click on upper right cog wheel)
  7. Click on List settings
  8. Under Advanced settings, turn off attachments
  9. Click Ok
  10. Back to the list

  11. Add columns as follows:

    Col Type Name Default Comments
    Text Title Already exists
    Multiline text Description What the course is about
    Managed Metadata Keywords Select Power Platform for Term set
    Expand More options... then...
    Turn on Allow multiple values
    You decide whether required or not, default no
