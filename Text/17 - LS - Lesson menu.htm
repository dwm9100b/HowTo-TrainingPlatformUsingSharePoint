We will be using a gallery control to create our left menu.

  1. We will be working with the conLessonMenu container
  2. So make sure the container is selected in the Tree view
  3. Then from the top toolbar, click on +Insert
  4. Search for and select the Blank Vertical gallery control
  5. When prompted for data source, select the colCourseLessons
  6. In the right properties pane, under Layout, select Title
  7. In the property above Layout, click on Edit of Fields
  8. I suspect the Title has already defaulted to Title, if not do so
  9. Rename your gallery to: galLessonMenu
  10. Now, before we go further, click on the Home Screen
  11. Then hold your Alt key down and click on any course
  12. This should populate some of the data structures
  13. Now click back on the Lessons Screen
  14. In Tree view expand the gallery control, galLessonMenu
  15. Rename Title: lbLessonlTitle
  16. Delete the Next Arrow icon
  17. Rename the Separator: rectLessonSeparator
  18. Move lblLessonSeparator to the bottom, via the â€¦ menu, reorder, Send to back
  19. Click on lblLessonTitle under the galLessonMenu control in the Tree view
  20. From the top toolbar, click on +Insert
  21. Expand the Icons section, scroll and select the Check icon
  22. Rename the icon: icoLessonCompleted
  23. Don't worry yet about properties for any of these controls, we'll get to them
  24. Click on the icoLessonCompleted control in galLessonMenu
  25. From the top toolbar, click on +Insert
  26. Select Text label
  27. Rename the Text label: lblLessonTime
  28. Make sure lblLessonTime is selected
  29. Again, from the top toolbar, click on +Insert
  30. Under the Icons section, select the Add icon
  31. Rename this to: icoLessonCursor
  32. Your controls within galLessonMenu should be in the following order
    • icoLessonCursor
    • lblLessonTime
    • icoLessonCompleted
    • lblLessonTitle
    • rectLessonSeparator

  33. Ok let's go update some of the properties for each control
    First up, properties for galLessonMenu, click on the galLessonMenu control

    Property Value
    X 0
    Y 0
    Width Parent.Width - 10
    TemplateSize 50
    TemplateFill If(ThisItem.IsSelected, Color.LightGray, Color.White)
    OnSelect 
    // Set our global html content to empty
// This should reset the scroll position, this is a hack
Set(gblHtmContent, "");

// Now start our timer, when it ends it will put selected
// html content into the global variable for display
Set(gblStartTimer, true);
    LoadingSpinner LoadingSpinner.None
    Height Parent.Height
    AlignInContainer AlignInContainer.Start

  34. Now let's update the properties of icoLessonCursor

    Property Value
    X 0
    Y 0
    Width Parent.TemplateWidth
    Height Parent.TemplateHeight
    Color Color.Transparent

  35. Now properties for icoLessonCompleted, click on the icoLessonCompleted control

    Property Value
    X 5
    Y 0
    Width Hard coded at 25
    Visible ThisItem.Completed
    Height Parent.TemplateHeight
    Color Color.Green

  36. Let's update the properties for lblLessonTime, click on the lblLessonTime control

    Property Value
    X Parent.TemplateWidth - Self.Width - 3
    Y 0
    Wrap false
    Width 53
    Visible ThisItem.Time > 0
    Text ThisItem.Time & " min"
    Size 8
    Height Parent.TemplateHeight
    Align Align.Right

  37. Next let's update properties for lblLessonTitle, click on the lblLessonTitle control

    Property Value
    X icoLessonCompleted.X + icoLessonCompleted.Width + 5
    Y 0
    Wrap false
    Width lblLessonTime.X - Self.X
    VerticalAlign VerticalAlign.Middle
    Text ThisItem.Title
    Size 11
    Height Parent.TemplateHeight

  38. Final step on this gallery section is to add a divider between the gallery and the content
  39. Make sure you have conLessonMenu selected in the Tree view
  40. Then from the top toolbar, click on +Insert
  41. Locate and select Rectangle, under Shapes section
  42. Rename this control: rectVerticalDivider
  43. Update the properties for rectVerticalDivider, click on the rectVerticalDivider control

    Property Value
    X 0
    Y 0
    Width 2
    Height Parent.Height
    Fill RGBA(56, 96, 178, 1)

  44. Good time to not only save the app, but also do a publish so you can revert back if needed
