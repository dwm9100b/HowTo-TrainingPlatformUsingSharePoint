This lesson we will get into the visual part of the galCourseMenu control.

  1. Over in the Tree view, click on the galCourseMenu control
  2. We added it in as a Blank vertial gallery, lets add our visuals
  3. Over in the right properties panel
  4. At the Layout property, change to Title and subtitle
  5. Expand the galCourseMenu to see all its sub controls
  6. Delete the NextArrow control
  7. Click on the Subtitle control, send it to the back
  8. Rename Subtitle to: lblCourseDesc
  9. Click on the Separator control, also send it to the back
  10. Rename Separator to: rectCourseSeparator
  11. Rename Title to: lblCourseTitle
  12. Make sure lblCourseTitle is still selected
  13. Then from the top toolbar, +Insert, under Icons, select Half filled circle
  14. Rename this: icoCourseInProgress
  15. With icoCourseInProgress still selected
  16. Then from the top toolbar, +Insert, under Icons, select Check (badge)
  17. Rename this: icoCourseCompleted
  18. With icoCourseCompleted still selected
  19. From the top toolbar, +Insert, under Icons, select Add
  20. Rename this: icoCourseCursor
  21. So from top to bottom your controls should be in this order:
    • icoCourseCursor
    • icoCourseCompleted
    • icoCourseInProgress
    • lblCourseTitle
    • lblCourseDesc
    • rectCourseSeparator

  22. Before going further, go ahead and right click on App and Run OnStart

  23. Controls are in place, now let's update their properties
    First up, click on icoCourseCursor and update the following properties

    Property Value
    X 0
    Y 0
    Width Parent.TemplateWidth
    Height Parent.TemplateHeight
    Fill Color.Transparent
    Color Color.Transparent

  24. Next click on icoCourseCompleted and update its properties

    Property Value
    X 10
    Y 0
    Width 21
    Visible 
    gblCourseDataLoaded && !IsBlank(
    LookUp(
        LessonsCompleted,
        CourseId = ThisItem.ID && LessonId = 0 && Student.Email = User().Email
    )
)
    Height 50
    Fill Color.Transparent
    Color Color.Green

  25. Next click on icoCourseInProgress and update its properties

    Property Value
    X icoCourseCompleted.X
    Y icoCourseCompleted.Y
    Width icoCourseCompleted.Width
    Visible 
    gblCourseDataLoaded && (IsBlank(
    LookUp(
        LessonsCompleted,
        CourseId = ThisItem.ID && LessonId = 0 && Student.Email = User().Email
    )
) && 
!IsBlank(
    LookUp(
        LessonsCompleted,
        CourseId = ThisItem.ID && LessonId <> 0 && Student.Email = User().Email
    )
))
    Height icoCourseCompleted.Height
    Fill Color.Transparent
    Color Color.Orange

  26. Next click on lblCourseTitle and update the following properties

    Property Value
    X icoCourseCompleted.X + icoCourseCompleted.Width + 10
    Y icoCourseCompleted.Y
    Width Parent.Width - (icoCourseCompleted.X + icoCourseCompleted.Width + Self.X)
    VerticalAlign VerticalAlign.Middle
    Text ThisItem.Title
    Size 18
    Height icoCourseCompleted.Height

  27. Finally click on lblCourseDesc and update its properties

    Property Value
    X lblCourseTitle.X
    Y lblCourseTitle.Y + lblCourseTitle.Height
    Width lblCourseTitle.Width
    Text ThisItem.Description
    Size 11
    Height Parent.TemplateHeight - Self.Y - 10
    AutoHeight false
