This application will be using the Term store of SharePoint for our keywords for filtering of courses. This does require that you have permissions to be able to add/edit terms.

  1. Navigate to the SharePoint admin center
  2. Expand the Content services area
  3. Click on Term store
  4. If you can, add yourself to the Admins section
  5. Once you have access to adding terms, proceed
  6. Click on the 3 vertical dot menu to the right of Taxonomy
  7. Select Add term group
  8. Name it: Training
  9. Then click on the 3 vertical dots for Training
  10. Click on Add term set
  11. Name it: Power Platform
  12. Now we can add our terms
  13. Click on the 3 vertical dots on Power Platform
    to repeatedly add the following terms:
    • App
    • Power App
    • Flow
    • Power Automate
    • Local storage
    • Remote storage
    • Function
    • SharePoint
    • Dataverse
    • Tutorial

Great! Off we go to create our lists now...
