Making great progress! This lesson we will create where the lesson content will be displayed.

  1. Over in the left Tree view, expand the Lessons Screen
  2. Click on conLessonContent
  3. From the top toolbar, click on +Insert
  4. Locate and select the Timer control, found under the Input section
  5. Rename this: timHtmScrollFix
  6. And again, from the top toolbar, click on +Insert
  7. Locate and select the HTML text control, found under the Display section
  8. Lets go ahead and rename it to: htmTemplate
  9. Update the htmTemplate control properties as follows

    Property Value
    X Parent.PaddingLeft
    Y 0
    Width Parent.Width - (Parent.PaddingLeft + Parent.PaddingRight)
    Size Leave default of 11, you can adjust later if you need
    HtmlText gblHtmContent
    Height Parent.Height

  10. Let's work on the timer control. Due to fact we cannot reset the scroll
    location of the Html text control, we have a hack that will reset to
    the top. So click on the timer control, timHtmScrollFix
  11. Update its properties as follows

    Property Value
    X 0
    Y 0
    Visible false
    Start gblStartTimer
    OnTimerEnd 
    // Reset our global that starts the timer
Set(gblStartTimer, false);
                            
// Now set our global active html variable with content
Set(gblHtmContent, galLessonMenu.Selected.Content);
    Duration 100
