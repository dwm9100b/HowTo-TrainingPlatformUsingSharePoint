App created, data sources added... lets update some App properties.

  1. Open up the Training Platform app at PowerApps.com
  2. Over in the Tree view, left side of the screen
  3. Click on the App object to select
  4. In the properties drop down, above Tree view
  5. Locate and select the StartScreen property
  6. In the fx (formula) bar add 'Home Screen'
  7. Again in the properties drop down, select OnStart
  8. Update OnStart with the following code:
    
// Start of App OnStart code

// Lets keep track of the app version
Set(gblAppVersion, "v0.001");

// Need a onetime variable for delaying the display
// of course compleded/in progress icons
Set(gblCourseDataLoaded, false);

// Initialize a blank collection for user selected keywords
ClearCollect(colFilterKeywords, Blank());

// Get a read only cache of all courses
ClearCollect(colAllCourses, 
    Sort(
        ShowColumns(Courses,
            "ID","Title","Description","TaxKeyword"
        ),
        Title
    )
);

// Initialize a variable for filterable metadata keywords
ClearCollect(colUniqueKeywords, {});

// For all the cached courses, get all the metadata keywords
// Add them to our unique keywords collection
ForAll(
    colAllCourses,
    ForAll(
        TaxKeyword,
        Collect(colUniqueKeywords, {Value: Label})
    )
);

// Now remove the duplicate keywords and sort by name, ascending
ClearCollect(colUniqueKeywords, Sort(Distinct(colUniqueKeywords, Value), Value));

// End of OnStart code
