For this application we will only be using 2 screens.

  1. Open up the Training Platform app at PowerApps.com
  2. You should already have renamed the default screen to Home Screen
  3. Now go ahead and add another New blank screen
  4. Name this one: Lessons Screen
  5. Then click on the Home Screen

  6. Update the following properties for Home Screen

    Property Value
    OnVisible // Initialize and make sure our global htm variable
    // resets to blank when returning to the home screen
    Set(gblHtmContent, "");
    Fill RGBA(56, 96, 178, 1)
