Let's work on the intro section. This will hold our descriptive text on what this app is about.

  1. In the left Tree view, click on conIntro
  2. From the top toolbar, click on +Insert
  3. Locate and select the HTML text control under Display
  4. Rename this: htmIntroContent
  5. Click on +Insert again
  6. Locate and select the Text label control
  7. Rename this: lblIntroTitle
  8. Again, click on +Insert
  9. Locate and select the Rectangle shape
  10. Rename this: rectIntroTitleCursor
  11. Click on +Insert
  12. Locate and select the Text label control
  13. Rename this: lblAppVersion
  14. Move this to the bottom, click on ... menu, reorder, send to back
  15. One last control, click on +Insert
  16. Locate and select the Timer control
  17. Rename this: timIconDelay
  18. Move this to the bottom, click on ... menu, reorder, send to back
  19. Your controls should be in the following order:
    • rectIntroTitleCursor
    • lblIntroTitle
    • htmIntroContent
    • lblAppVersion
    • timIconDelay
  20. Now that we have our controls, lets update their properties

  21. First up, click on rectIntroTitleCursor, update as follows

    Property Value
    X Parent.X
    Y Parent.Y
    Width Parent.Width
    Height 87
    Fill Color.Transparent

  22. Next, click on lblIntroTitle and update as follows

    Property Value
    X rectIntroTitleCursor.X
    Y rectIntroTitleCursor.Y
    Width rectIntroTitleCursor.Width
    Text "Training Platform"
    Size 20
    Height rectIntroTitleCursor.Height
    Color Color.White
    Align Align.Center

  23. Click on htmIntroContent and update its properties

    Property Value
    X rectIntroTitleCursor.X
    Y rectIntroTitleCursor.Height
    Width rectIntroTitleCursor.Width
    Size 18
    PaddingRight 50
    PaddingLeft 50
    HtmlText You will add your own intro html text here
    Height Parent.Height - Self.Y
    Color Color.White

  24. Click on lblAppVersion and update its properties

    Property Value
    X htmIntroContent.X
    Y App.Height - Self.Height
    Width 150
    Text gblAppVersion
    Size 10
    PaddingLeft 50
    Color Color.White

  25. Click on timIconDelay and update its properties

    Property Value
    X lblAppVersion.X
    Y lblAppVersion.Y
    Width lblAppVersion.Width
    Visible false
    Start !gblCourseDataLoaded
    OnTimerEnd Set(gblCourseDataLoaded, true)
    Height lblAppVersion.Height
    Duration 1500
