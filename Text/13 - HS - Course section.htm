Time to build out our course selection section. We will be adding the controls and then we'll come back and update their properties.

  1. Before we go further, right click on the App object
  2. Now click on Run OnStart
  3. This will populate some data we'll use
  4. Now in the Tree view, click on conCourses
  5. From the top toolbar, click on +Insert
  6. Locate and select the Blank Vertical gallery under Layout
  7. When prompted, select colAllCourses as the data source
  8. Rename this: galCourseMenu
  9. Again, from the top toolbar, click on +Insert
  10. Locate and select the Combo box under Input
  11. When prompted, select colUniqueKeywords as the data source
  12. Rename this: cboFilterTags
  13. Again, from the top toolbar, click on +Insert
  14. Locate and select the Text label control
  15. Rename this: lblSelectCourse
  16. From the top toolbar, click on +Insert
  17. Locate and select the Rectangle shape under Shapes
  18. Rename this: rectSelectCourseCursor
  19. The order of the controls from top to bottom should be
    • rectSelectCourseCursor
    • lblSelectCourse
    • cboFilterTags
    • galCourseMenu

  20. Time to update some properties
  21. First up, click on rectSelectCourseCursor and update the following

    Property Value
    X 0
    Y 0
    Width Parent.Width
    Height rectIntroTitleCursor.Height
    Fill Color.Transparent

  22. Next let's update properties for lblSelectCourse, click on it

    Property Value
    X rectSelectCourseCursor.X
    Y rectSelectCourseCursor.Y
    Width rectSelectCourseCursor.Width
    Text "Select a Course"
    Size 20
    Height rectSelectCourseCursor.Height
    Color Color.White
    Align Align.Center

  23. Now click on cboFilterTags and update the following properties

    Property Value
    X (Parent.Width / 2) - (Self.Width / 2)
    Y lblSelectCourse.Y + lblSelectCourse.Height
    Width Parent.Width - 150
    SelectMultiple true
    OnChange // Every time there is a change to the combo box entries
    // add any selected items to our search tag collection
     ClearCollect(colFilterTags, Self.SelectedItems)
    NoSelectionText "You can filter the list here..."
    Height 40
    FocusedBorderThickness 0
    BorderThickness 1

  24. And finally, click on galCourseMenu and update the following properties
    Note that we are only updating visual properties here, next lesson is functional

    Property Value
    X cboFilterTags.X
    Y cboFilterTags.Y + cboFilterTags.Height + 10
    Width cboFilterTags.Width
    TemplateSize 145
    TemplatePadding 0
    TemplateFill If(ThisItem.IsSelected, Color.LightGray, Color.White)
    LoadingSpinner LoadingSpinner.None
    Height Parent.Height - Self.Y - 40
    Fill Color.White
    BorderThickness 1
    BorderStyle BorderStyle.Solid
