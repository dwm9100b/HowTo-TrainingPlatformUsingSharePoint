Now that our Home Screen is working, let's work on the Lessons Screen.
In this lesson we will be creating the layout where controls will be added.

  1. Make sure the Lessons Screen is active, click on it
  2. Now we will add a container for the top screen header
  3. From the top toolbar, click on +Insert
  4. Locate and select the Container control under Layout, not horizontal or vertical
  5. Rename this to: conScreenHeader
  6. Let's change some properties, make sure to click on conScreenHeader

    Property Value
    X 0
    Y 0
    Width App.Width
    Height 55
    Fill RGBA(56,96,178,1)

  7. Next up will be a container that will hold our left gallery menu
  8. Make sure to first click on the Lessons Screen screen control
  9. Then from the top toolbar, click on +Insert
  10. Locate and select the Horizontal container control
  11. Rename this one to: conLessonMenu
  12. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  13. Change the following properties for conLessonMenu

    Property Value
    X conScreenHeader.X
    Y conScreenHeader.Y + conScreenHeader.Height
    Width Hard coding the value to 375
    Height App.Height - Self.Y

  14. Next up will be a container that will hold our Lesson header
  15. Again, make sure you click on the Lessons Screen screen control first
  16. Then from the top toolbar, click on +Insert
  17. Locate and select the Container control, not horizontal or vertical
  18. Rename this one to: conLessonHeader
  19. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  20. Change the following properties for conLessonHeader

    Property Value
    X conLessonMenu.X + conLessonMenu.Width
    Y conLessonMenu.Y
    Width App.Width - Self.X
    PaddingRight 15
    PaddingLeft 10
    Height 80

  21. Next up will be a container that will hold the controls for the content
  22. Make sure to click on the Lessons Screen screen control first
  23. From the top toolbar, click on +Insert
  24. Locate and select just the Container control, not vertical or horizontal
  25. Rename this one to: conLessonContent
  26. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  27. Change the following properties for conLessonContent

    Property Value
    X conLessonHeader.X
    Y conLessonHeader.Y + conLessonHeader.Height
    Width conLessonHeader.Width
    PaddingRight 15
    PaddingLeft 10
    Height App.Height - Self.Y

  28. Now move/rearrange your containers in the Tree view a bit more logical as follows
    1. conScreenHeader
    2. conLessonMenu
    3. conLessonHeader
    4. conLessonContent

  29. One last quick update to a screen property, click on Lessons Screen
  30. In the properties drop down, select the OnVisible property
  31. Add the following code
    // Gallery will automatically default to first item when screen loads
// Since we have to use a hack, we manually load the
// initial html content into our global variable
Set(gblHtmContent, First(colCourseLessons).Content);

// Initialize our global for timer start/stop
// Which we use for loading new html content
Set(gblStartTimer, false);

// End of the OnVisible code
  32. Good spot to save your app
