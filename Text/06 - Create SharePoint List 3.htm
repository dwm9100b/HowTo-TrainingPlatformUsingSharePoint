Almost done with our SharePoint lists. This final list will hold status of completion. This list will hold dual purpose, completion status of lessons and whether a course has been compeleted.

  1. Again, if not already there, navigate to the Training site
  2. From the upper toolbar, click on +New, select List
  3. Create a new Blank list
  4. Name: LessonsCompleted
  5. Click on Create
  6. Once created, go into Settings (click on upper right cog wheel)
  7. Click on List settings
  8. Under Advanced settings, turn off attachments
  9. Click Ok
  10. Back at the list settings, click on the Title column
  11. Change the Required setting to No
  12. Click on Ok
  13. Now back to the list

  14. Add columns as follows:

    Col Type Name Default Comments
    Number CourseId Decimal places: 0
    Under More options...
    Thousands separator: No
    Required: Yes
    Number LessonId Decimal places: 0
    Under More options...
    Thousands separator: No
    Required: No
    Person Student Allow groups: No
    Under More options...
    Required: Yes

  15. Before we finish this lesson, one final thing to do
  16. Click on the Title column
  17. From the drop down hover over Column settings
  18. Click on Show/hide columns
  19. Remove the check by Title, then click on Apply
