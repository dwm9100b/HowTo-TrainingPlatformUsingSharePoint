Now that most functionality is present, lets update our screen header.

  1. Select the control conScreenHeader in the Tree view
  2. Then from the top toolbar, click on +Insert
  3. Locate and select the Text label control
  4. Rename this one: lblMinsRemaining
  5. From the top toolbar again, click on +Insert
  6. Locate and select the Text label control
  7. Rename this one: lblAppTitle
  8. Again, click on +Insert
  9. Locate and select the Text label control
  10. Rename this one: lblPctComplete
  11. And again, click on +Insert
  12. Locate and select the Rectangle shape, under Shapes
  13. Rename this one: rectScreenHeaderCursor
  14. And finally, click on +Insert
  15. Locate and select under Icons, the Left icon
  16. Rename this one: icoGoBack
  17. Your controls should be in the following top to bottom order
    • icoGoBack
    • rectScreenHeaderCursor
    • lblPctComplete
    • lblAppTitle
    • lblMinsRemaining

  18. Now select the 3 text labels
  19. We can now edit common properties

    Property Value
    Y 0
    Width 225
    VerticalAlign VerticalAlign.Middle
    Size 13
    Height Parent.Height
    Color Color.White

  20. We will start on the icoGoBack control, click on it
  21. Update its properties as follows

    Property Value
    X 5
    Width 40
    PaddingRight 15
    OnSelect Navigate('Home Screen', ScreenTransition.Fade);
    Icon Icon.ChevronLeft
    Color Color.White

  22. Now go ahead and click on the rectScreenHeaderCursor
  23. Update its properties as follows

    Property Value
    X Parent.X
    Y Parent.Y
    Width Parent.Width
    Height Parent.Height
    Fill Color.Transparent

  24. Next, let's work on lblPctComplete, in the Tree view, click it
  25. Update its properties as follows

    Property Value
    X icoGoBack.X + icoGoBack.Width
    Text 
    RoundUp( 
     If(CountIf(colCourseLessons,Completed) > 0, 
  
         // If there are items completed, Count them then
         // divide that by the count of items that have minutes
         (CountIf(colCourseLessons,Completed) /
         (CountIf(colCourseLessons,Time > 0)) * 100),
  
          
         // No items completed, means 0
         0
     ),
  
      
     // Round up to 0 decimal places
     0 
  
 ) & "% completed"

  26. Now click on lblAppTitle in the Tree view
  27. Update its properties as follows

    Property Value
    X (App.Width - Self.Width) / 2
    Width 450
    Text galCourseMenu.Selected.Title
    Align Align.Center

  28. Last up, click on lblMinsRemaining
  29. Update its properties as follows

    Property Value
    X App.Width - Self.Width - 20
    Text Sum(colCourseLessons, Time) - Sum(Filter(colCourseLessons, Completed), Time) & " minutes remaining"
    Align Align.Right

  30. Our screen header is done
