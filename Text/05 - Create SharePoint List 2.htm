Now for our second list, this one will hold the actual lessons.

  1. If not already there, navigate to the Training site
  2. From the upper toolbar, click on +New, select List
  3. Create a new Blank list
  4. Name: CourseLessons
  5. Click on Create
  6. Once created, go into Settings (click on upper right cog wheel)
  7. Click on List settings
  8. Under Advanced settings, turn off attachments
  9. Click Ok
  10. Back to the list

  11. Add columns as follows:

    Col Type Name Default Comments
    Text Title Always part of a list
    Lookup Course Source: Courses
    Column: Title
    Under More options...
    Add addl columns, select ID
    Required: Yes
    Number LessonId Decimal places: 0
    Under More options...
    Thousands separator: No
    Required: Yes
    Number Time 0 Decimal places: 0
    Under More options...
    Thousands separator: No
    Required: Yes
    Multiline text Content Under More options...
    Use rich text: Yes

  12. Let's move some columns around in the list
  13. Drag the Title column and put it right between Time and Content
  14. Also a good time to adjust your column widths per your liking
  15. Now click on +New to add a new list item, we'll edit the form
  16. Towards the upper right, click on the Edit form icon
  17. Select Edit columns
  18. Move the columns around so the order is as follows
    • Course
    • LessonId
    • Time
    • Title
    • Content
