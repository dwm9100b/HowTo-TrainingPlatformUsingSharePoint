Now let's add our SharePoint lists as data sources.

  1. You should still be in the Training Platform app
  2. If you have not added any connectors yet, thats first up
  3. Top toolbar, click on Add data
  4. Scroll down to Connectors
  5. Locate and select SharePoint
  6. Login as per prompts
  7. When prompted for site, select Training
  8. When prompted to select lists add the following:
    • Courses
    • CourseLessons
    • LessonsCompleted
  9. Make sure to click on Connect towards the bottom

Done... data connections have been established
